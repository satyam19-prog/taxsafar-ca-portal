## TaxSafar - CA Firm Inquiry System

### Admin Credentials
- Email: admin@taxsafar.com
- Password: Admin@123

### Setup
1. Import `taxsafar.sql` into MySQL
2. Edit `includes/db.php` with your DB credentials
3. Place project in `htdocs/` or `www/`
4. Visit `http://localhost/taxsafar-assignment/`
5. Admin panel: `http://localhost/taxsafar-assignment/admin/login.php`

### Tech Stack
- PHP 8+ (Core, no frameworks)
- MySQL with PDO + Prepared Statements
- Vanilla HTML/CSS/JS
